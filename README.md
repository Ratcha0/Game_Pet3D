# 🐱 LikeGotchi 3D: คู่มือเลี้ยงน้องฉบับเข้าใจง่าย 🐾✨

ยินดีต้อนรับสู่ **LikeGotchi** เกมเลี้ยงสัตว์ 3 มิติมิติใหม่ที่คุณสามารถพกพาไปได้ทุกที่บนหน้าเว็บ! ไม่ว่าคุณจะเป็นสายเกมเมอร์หรือคนรักสัตว์ทั่วไป คู่มือนี้จะช่วยให้คุณดูแลน้องได้อย่างมีความสุขครับ

---

## 🚀 1. เริ่มเล่นยังไง? (How to Start)

เกมของเราใช้ระบบ **"ไอดีส่วนตัวผ่านลิงก์"** ครับ แค่คุณเติมชื่อตัวเองต่อท้ายลิงก์ คุณก็จะมีน้องแมวเป็นของตัวเองทันที!
*   **ตัวอย่าง:** `http://.../?userId=ชื่อของคุณ`
*   **ล็อครหัสผ่าน:** เมื่อเข้าครั้งแรก ระบบจะให้คุณตั้ง **รหัส 4 หลัก** เพื่อป้องกันไม่ให้คนอื่นแอบเข้ามาเลี้ยงน้องแมวของคุณครับ 🔒

---

## 🍖 2. หน้าที่ของคุณในฐานะเจ้าของ (Common Actions)

ในหน้าจอจะมีปุ่มหลักๆ 4 ปุ่มที่คุณต้องหมั่นกดดูแต่น้องครับ:

| ปุ่มกิจกรรม | ทำแล้วได้อะไร? | ผลลัพธ์ที่ตามมา |
| :--- | :--- | :--- |
| **🍖 Feed (ป้อนอาหาร)** | น้องจะอิ่มและโตไวขึ้น | ได้ค่า **XP** (ประสบการณ์) เพื่อเลเวลอัป |
| **🧼 Clean (อาบน้ำ)** | น้องจะตัวสะอาด สดชื่น | เพิ่มค่า **Clean** และทำให้น้องอารมณ์ดี |
| **🎾 Play (เล่นด้วย)** | น้องจะรักคุณมากขึ้น | เพิ่มค่า **Love** ได้ไวที่สุด |
| **💩 Scoop (เก็บอึ)** | เมื่อเห็น "อึ" บนพื้น ให้รีบเก็บ | **ได้รับเหรียญ (Tokens)** และคะแนนสะสม |

---

## 🪙 3. วิธีหาเหรียญและรางวัล (Earning Rewards)

อยากได้เหรียญไปช้อปปิ้ง? นี่คือวิธีหาเงินในเกมครับ:

*   **🧹 เก็บกวาดกองอึ:** ทุกครั้งที่น้องอึทิ้งไว้บนพื้น อย่าปล่อยผ่าน! จิ้มเก็บแล้วคุณจะได้ **เหรียญสีเงิน** ทันทีครับ (บางครั้งอาจโชคดีได้โบนัสก้อนใหญ่ถ้าเจออึสีทอง!)
*   **🎁 ของขวัญจากความสุข:** ถ้าน้องแมวมีค่าความรัก (Love) สูงๆ น้องจะสุ่มทำ **เหรียญทองหล่นไว้บนพื้น** ให้คุณเก็บฟรีๆ เป็นระยะครับ
*   **🆙 รางวัลเลเวลอัป:** เมื่อน้องแมวโตขึ้น (Level Up) คุณจะได้รับ **เหรียญก้อนใหญ่** เป็นรางวัลความตั้งใจครับ
*   **📜 ภารกิจรายวัน:** กดดูที่รูป **ม้วนกระดาษ** ทางขวาบน ถ้าทำครบตามที่บอก (เช่น ป้อนอาหารครบ 3 ครั้ง) รับเหรียญรางวัลพิเศษไปเลย!

---

## ⚡ 4. พลังงานและร้านค้า (Stamina & Shop)

*   **⚡ พลังงาน:** น้องแมวต้องใช้พลังงานในการทำกิจกรรม ถ้าพลังงานหมด น้องจะเหนื่อยและทำกิจกรรมไม่ได้
*   **🛒 ร้านค้า (Shop):** นำเหรียญ (Tokens) ที่สะสมมาได้ ไปซื้อ **"ขนมเพิ่มพลังงาน"** ในร้านรูปรถเข็น เพื่อให้น้องกลับมาวิ่งเล่นได้ต่อครับ
