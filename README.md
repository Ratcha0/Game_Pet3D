# 🐱 LikeGotchi: 3D Gamified Pet Simulator Widget 🐾✨

ยินดีต้อนรับสู่โปรเจกต์ **LikeGotchi** เกมเลี้ยงสัตว์ 3 มิติประสิทธิภาพสูงที่ออกแบบมาเพื่อใช้งานเป็น **Web Widget** สำหรับ Dashboard ขนาดเล็ก (316x182px) และหน้าจอ Mobile Full Screen ที่เน้นความพรีเมียม สวยงาม และเล่นได้จริง 100%

---

## 🌟 จุดเด่นของโปรเจกต์ (Key Features)

### 🎮 ระบบการเล่น (Gameplay Dynamics)
*   **3D Interactive World:** พัฒนาด้วย **Three.js** รองรับการเดินแบบกึ่งอัตโนมัติ (Auto-Walk) และการแตะที่พื้นเพื่อสั่งให้เดิน (Tap to Walk)
*   **Multi-Template Support:** สามารถสลับโหมดการเล่นได้ 3 รูปแบบผ่านหน้า Admin:
    *   **🐱 Pet (สัตว์เลี้ยง):** เน้นการป้อนอาหาร อาบน้ำ และเล่นสนุก
    *   **🚗 Car (ยานพาหนะ):** เน้นการเติมน้ำมัน การล้างรถ และการซ่อมบำรุง
    *   **🌱 Plant (ต้นไม้):** เน้นการรดน้ำ การใส่ปุ๋ย และการพรวนดิน
*   **Real-time Simulation:** ค่าสถานะ **หิว (Hunger)**, **ความรัก (Love)**, **ความสะอาด (Clean)** และ **พลังงาน (Stamina)** จะลดลงและฟื้นฟูแบบเรียลไทม์แม้ไม่ได้เปิดหน้าจอค้างไว้

### 📱 การออกแบบ UI/UX (Aesthetic & Compact Design)
*   **Ultra-Compact HUD:** ปรับปรุง UI ให้เล็กลงแต่ยังใช้งานได้จริง (Pixel-Perfect) สำหรับพื้นที่ขนาดจำกัด 316px
*   **Floating Level Badge:** แสดงเลเวล (Lv.) บนดีไซน์ที่หรูหรา ไม่เกะกะสายตา
*   **Radar Navigation:** มีลูกศรชี้เป้าอัจฉริยะ (Indicator) ช่วยบอกทิศทางของไอเทมที่ดรอปอยู่นอกจอ

### 🛠️ ระบบหลังบ้าน (Advanced Admin Dashboard)
*   **Live Preview Interface:** หน้าต่าง Admin ที่สามารถเห็นการเปลี่ยนแปลงทั้งในรูปแบบ Card และ Full Screen ได้ทันทีโดยไม่ต้องกดบันทึก
*   **Logic Presets:** เลือกปรับระดับความยาก (Normal/Hard) และความจุของกระเป๋าพลังงานได้ตามต้องการ

## 🕹️ วิธีการเล่น (How to Play)

### 1. ดูแลสถานะพื้นฐาน (Caring)
*   **🍖 Feed (ป้อนอาหาร):** ฟื้นฟูค่าความหิวและขยับเลเวล XP
*   **🧼 Clean (ทำความสะอาด):** รักษาความสะอาดให้น้องสดชนตลอดเวลา
*   **💩 Scoop Poop (เก็บอึ/ซ่อมบำรุง):** คลิกที่ก้อนอึบนพื้น (หรือปุ่ม UI) เพื่อทำความสะอาดและลุ้นรับรางวัลแรร์
*   **🎾 Play (เล่นด้วย):** เพิ่มค่าความรัก (Love) ได้ไวที่สุด และเพิ่มค่าความสุขโดยรวม

### 2. อีเวนต์สิทธิพิเศษ (Bonus Events)
*   **🔥 Fever Mode:** รักษาค่าสถานะหลักให้เกิน **80%** พร้อมกัน เพื่อเข้าสู่โหมด "สมบูรณ์แบบ" ทำให้พลังงานฟื้นฟูเร็วขึ้น 1.5 เท่า
*   **🪙 Happy Coins:** สัตว์เลี้ยงที่อารมณ์ดีมาก (>80%) จะสุ่มดรอปเหรียญทองให้ที่พื้นเป็นระยะๆ
*   **🆕 Level Up:** สะสม XP จนเต็มหลอดเพื่ออัปเลเวล รับคะแนนอภิมหาศาลและเติมพลังงานจนเต็มถังทันที

### 3. ตลาดและการซื้อของ (Shop & Quests)
*   **📜 Daily Quests:** ทำภารกิจรายวันให้ครบ (เช่น ป้อนอาหาร 3 ครั้ง) เพื่อรับโบนัส Score และ Tokens พิเศษ
*   **🛒 Shop:** ใช้เหรียญ (Tokens) ที่สะสมได้จากการเก็บอึหรือดรอปจากความสุข มาซื้อแพ็กเกจเพิ่มพลังงาน (⚡)

---

## 📦 รายการไอเทมดรอป (Drop Items)
*   **💩 ก้อนอึ / ของร่วง:** เก็บเพื่อเพิ่มความสะอาดและลุ้นรับเหรียญทอง (Rare Drop 10%)
*   **🪙 เหรียญทอง:** ของรางวัลฟรีที่เกิดขึ้นเมื่อน้องมีความสุข ให้คุณเก็บสะสมเพื่อไปช้อปปิ้งต่อ

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
*   **Core:** HTML5, Javascript (ES6+)
*   **Engine:** Three.js (3D Rendering)
*   **Styling:** Vanilla CSS, Tailwind CSS (Responsive Design)
*   **Storage:** LocalStorage (Offline Persistent Save)
